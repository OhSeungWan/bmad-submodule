# Getting Started with DAE

DAE (Data Analysis Expert)에 오신 것을 환영합니다! 이 가이드를 통해 빠르게 시작할 수 있습니다.

---

## What This Module Does

DAE는 데이터 분석을 모르는 사용자도 쉽게 데이터 기반 의사결정을 할 수 있도록 돕는 AI 에이전트 모듈입니다.

**핵심 기능:**
- Amplitude 데이터 실시간 조회 및 분석
- A/B 테스트 설계 및 결과 해석
- 퍼널 분석 및 인사이트 도출
- 이상 징후 선제적 감지

---

## Installation

모듈이 아직 설치되지 않았다면:

```bash
bmad install dae
```

설치 중 다음 질문에 답하세요:
- **데이터 분석 경험 수준** — 초급/중급/고급
- **분석 리포트 저장 위치** — 기본값 권장
- **선제적 위험 알림** — 활성화 권장

---

## First Steps

### 1. 서비스 컨텍스트 설정 (최초 1회)

재만에게 서비스의 퍼널 구조와 주요 지표를 알려주세요:

```
[SC] 또는 "서비스 컨텍스트 설정"
```

이 정보는 재만의 메모리에 저장되어 이후 분석에 활용됩니다.

### 2. 첫 데이터 분석

궁금한 것을 물어보세요:

```
[DA] 또는 "데이터 분석"

예시 질문:
- "지난 주 회원가입 전환율이 어떻게 됐어?"
- "최근 30일 DAU 추이를 알려줘"
- "결제 퍼널에서 어디서 가장 많이 이탈해?"
```

## Common Use Cases

### PM을 위한 사용 예시

```
"신규 기능 출시 후 사용률이 어떻게 변했어?"
"A/B 테스트 결과를 해석해줘"
"다음 분기에 어떤 퍼널을 개선해야 할까?"
```

### 개발자를 위한 사용 예시

```
"이 기능 변경이 전환율에 얼마나 영향을 줄까?"
"배포 후 이상 징후가 있는지 확인해줘"
"에러 발생률이 평소와 다른지 체크해줘"
```

### 디자이너를 위한 사용 예시

```
"이 퍼널에서 어떤 실험이 있었어? 뭐가 효과적이야?"
"이전 UI 변경 실험 결과를 알려줘"
"사용자들이 어디서 가장 많이 헤매?"
```

### 비개발 직군을 위한 사용 예시

```
"지난 달 매출이 얼마야?"
"어떤 채널에서 가입이 많이 들어와?"
"캠페인 효과가 있었어?"
```
